# 🧩 ПОРТФЕЛЬ «YIELD & HODL» | ТЕХНОФИН 🔑

Личный on-chain фонд со стратегией динамической ребалансировки (30/30/40).

---

### 🏛️ Базовая стратегия инвестора: 3 главных вопроса

#### 1. ЧТО покупать?
Портфель собран из трёх категорий активов и готов к любым ситуациям на рынке:
*   **🟠 РОСТ (30%):** Bitcoin (BTC) - главный долгосрочный локомотив.
*   **🛡️ ЗАЩИТА (30%):** Tether Gold (XAUT) - токенизированное золото.
*   **🧲 YIELD (40%):** DeFi-стратегия - печатный станок реальной доходности (Real Yield).

#### 2. КОГДА покупать?
Стратегия **DCA (Dollar-Cost Averaging)**: покупка активов каждую неделю, по воскресеньям, ровно на **$50**. Вне зависимости от того, растёт рынок или падает. Дисциплина всегда бьёт тайминг.

#### 3. СКОЛЬКО покупать?
**Динамическая ребалансировка**. Каждое воскресенье мы берём прибыль от фарминга + $50 DCA, находим самый отставший от целевой пропорции актив в DropStab и называем его **«Активом Недели»**, куда заливаем весь свободный кэш. Рынок сам выравнивает наш фонд.

```markdown
[🧲 YIELD (Доходность)] + [💵 Еженедельный DCA (\$50)] ──> 🔍 АНАЛИЗ ОТКЛОНЕНИЙ ──> 🎯 НАПРАВЛЕНИЕ В «АКТИВ НЕДЕЛИ» ──> [🟠 РОСТ] / [🛡️ ЗАЩИТА] / [🧲 YIELD]
```
